## Getting and Cleaning Data

### What to do

+ Open up RStudio
+ Make sure that there is a folder called "UCI HAR Data" in the same directory as the R script.
+ Set the working directory to where the script is
+ Load the script
	source('run_analysis.R')
+ Run the analysis
	result <- run_analysis()
+ View the result
	View(result)
